# BoxBreathing
An easy App to help you take deep breaths with box breathing

## 运行步骤

1. git clone https://github.com/BillyYccc/BoxBreathing.git
2. cd BoxBreathing/
3. npm install
4. 打开Android模拟器或者连接真机打开调试模式
5. react-native run-android
